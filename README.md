# 🧩 Pattern Printing Practice – Mastering Loops & Logic

Welcome to my **Pattern Printing Practice Repository** 🎯

## ✨ What You'll Find Here

- 🔢 Number Patterns
- 🟩 Star (`*`) Patterns
- 🔺 Triangle, Pyramid, Diamond Shapes
